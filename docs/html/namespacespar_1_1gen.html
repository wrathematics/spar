)
inlinestatic

Generate a sparse matrix whose entries are randomly.

Parameters
[in]seedRandom seed.
[in]prop_denseProportion of density. Should be a number between 0 and 1. Other numbers will be truncated to 0 (those less than 0) or 1 (those greater than 1).
[in]nrows,ncolsThe dimensions of the return.
[in]exactShould the number of non-zero entries exactly match the prop_dense parameter? If true, a reservoir sampelr will be used. Otherwise each of the nrows*ncols entries will be treated as an independent Bernoulli trial.
Returns
A sparse matrix.

Memory Allocations
The return matrix is allocated/resized as necessary.

Exceptions
If a memory allocation fails, a bad_alloc exception will be thrown.
